Our robots like a game that require ability to calculate. In the latest journey they found a new game -- Numbered Triangles. In this game a player take 6 chips from the pile. Each chip is a equilateral triangle with three numbers on the edges. You can move, rotate and flip the chips so they form a hexagon. The hexagon is only legal if edges common to two triangles have the same number on them.

The score for a legal hexagon is the sum of the numbers on the outside six edges. The player's goal to find the highest score that can be achieved with the given six chips.

Each chip is represented as a list with three natural numbers. You should learn our robots to find the highest score for the given chips and return it as a number.

Input: The chips' info. A list of lists. Each list contain three natural numbers (int).

Output: The highest possible score (int).
